# Presentation Tips

* Keep each slide concise (no more than 5–7 bullet points).
* Use diagrams instead of long paragraphs.
* Demonstrate the application if possible.
* Maintain consistent fonts and colors.
* Rehearse to stay within the allocated time.
* Be prepared to explain AI workflow, database relationships, and architectural decisions.

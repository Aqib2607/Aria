# Title

**Aria: An AI-Powered Career Guidance and Learning Roadmap Assistant**

---

# 1. Introduction

Career planning has become increasingly complex due to the rapid evolution of technology, changing job market demands, and the vast amount of learning resources available online. Many students struggle to identify suitable career paths, understand the skills required for their desired professions, and organize an effective learning plan.

Aria is proposed as an AI-powered web application that addresses these challenges by providing personalized career recommendations, identifying skill gaps, generating structured learning roadmaps, recommending learning resources, and assisting users with interview preparation.

The platform combines Artificial Intelligence with modern web technologies to create an intelligent career development assistant for students, graduates, and lifelong learners.

---

# 2. Problem Statement

Many learners face several challenges:

* Lack of structured career guidance
* Difficulty selecting an appropriate career path
* Limited understanding of required technical skills
* No personalized learning roadmap
* Information overload from online learning platforms
* Limited interview preparation resources

Traditional career counseling is often expensive, inaccessible, or unable to provide personalized recommendations at scale.

---

# 3. Proposed Solution

Aria provides an AI-driven solution that:

* Recommends suitable career paths
* Analyzes existing skills
* Identifies missing competencies
* Generates personalized learning roadmaps
* Recommends learning resources
* Generates interview questions
* Tracks learning progress
* Provides actionable career insights

The application leverages Google Gemini AI through a Laravel backend to generate intelligent recommendations while maintaining user data securely in a MySQL database.

# 10. Feasibility Study

## Technical Feasibility

The selected technologies are mature, widely adopted, and well documented, making implementation practical within the project timeline.

## Economic Feasibility
